# Truss Component Library

The [Truss component library](https://github.com/ucsc/truss) is a set of reusable web components built with [Stencil.js](https://stenciljs.com). With Truss, you can quickly build branded, lightweight, and accessible web applications.

## Issues

Find a bug or request a new component by [submitting an issue](https://github.com/ucsc/truss/issues/new/).
